# Creditworthiness Prediction using Machine Learning
[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://github.com/codespaces/new?hide_repo_select=true&repo=HusnainQasid/credit-risk-predictor)


## Description
This project implements a Machine Learning model to predict the creditworthiness of an individual based on financial and demographic input features.  
The model analyzes the provided data and classifies applicants as **good** or **bad credit risks**, helping automate credit risk assessment.

---

## Features
- Data preprocessing for numeric and categorical features  
- Machine Learning model using **Logistic Regression**  
- Prediction of creditworthiness from user input  
- Model evaluation using accuracy and performance metrics  

---

## Technologies Used
- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Matplotlib  

---

## Workflow
1. Load and clean the dataset  
2. Encode categorical variables  
3. Scale numeric features  
4. Train the Logistic Regression model  
5. Evaluate model performance  
6. Predict creditworthiness for new input data
